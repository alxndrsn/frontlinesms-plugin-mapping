The Debug menu

In the Classic and New View, the Debug menu provides direct access to the FrontlineSMS database, and options to run SQL queries

Direct Database Access

Warning: This is an advanced feature and should ONLY be used by system administrators. Inadvertently altering settings at database level could render your FrontlineSMS installation unusable and non-recoverable. Use at your own risk!

For system administrators and programmers, FrontlineSMS allows direct access to the underlying database through the Direct Database Access menu. Here, administrators can study the database structure, view database entries, and run SQL queries
